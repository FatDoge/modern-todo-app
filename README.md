# Modern Todo App

一个具有现代化 UI 和流畅动画效果的待办事项应用。

## 特性

- 🎨 现代化 UI 设计
- ✨ 流畅的动画效果
- 📱 响应式布局
- 🔄 本地数据持久化
- 🎯 任务筛选功能
- ✅ 任务完成状态切换
- 🗑️ 删除任务功能
- 📊 任务统计信息

## 技术栈

- React
- TypeScript
- Tailwind CSS
- Vite
- Lucide Icons

## 开始使用

1. 克隆仓库
```bash
git clone https://github.com/FatDoge/modern-todo-app.git
cd modern-todo-app
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 http://localhost:5173

## 构建

```bash
npm run build
```
