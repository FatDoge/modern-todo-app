## Getting Started

1. Clone the repository
```bash
git clone https://github.com/FatDoge/modern-todo-app.git
cd modern-todo-app
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and visit http://localhost:3000

## Build

```bash
npm run build
```
